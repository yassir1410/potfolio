## 🌟 Overview

A modern, responsive portfolio website showcasing my journey as a Software Engineering Student at INSEA. Built with Next.js and Tailwind CSS, this portfolio demonstrates my expertise in Java, Spring Boot, Angular and modern web technologies.

## 🛠️ Quick Start

```bash
# Clone the repository
git clone https://github.com/yassir-fahimi/Personal-portfolio.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
